# SoftRenderer-Unity
在Unity中实现光栅化渲染只是为了深入理解光栅化渲染的流程和细节，使用的算法大多没有经过优化，使用到Unity的功能：<br>
    * CommandBuffer显示光栅化结果<br>
    * 数学计算模型<br>
    * 加载纹理加载<br>
需要注意的是：<br>
    * Unity中的世界空间和观察空间都是左手坐标系<br>
    * Unity中三角形顶点顺序为顺时针的当作正面<br>
    * Unity中向量是行矩阵<br>
